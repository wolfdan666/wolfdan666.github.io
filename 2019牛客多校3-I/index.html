题目

题目链接

Median

题意

3个数产生一个中位数，现在给你一串中位数，请还原出一个合理的原串

Input

T组,每组给n表示原串的长度,然后是给你中位数串b[1]->b[n-1]

范围: n的和不超过10^6,每个b不超过10^9

Output

有合理的串则输出原串,否则输出-1

题解

结论

若存在合理的解,那么解的每个位置的最终值一定是它能影响到的3个中位数之一
